# TextEditor
## 1. Необхідні інструменти та середовище
Для роботи з проєктом на комп'ютері має бути встановлене наступне програмне забезпечення:

### 1.1. Платформа .NET
Необхідно встановити .NET SDK версії 6.0 або новішої.
Перевірити наявність можна командою в терміналі: dotnet --version

### 1.2. Система контролю версій
Git (для клонування репозиторію).

### 1.3. Середовище розробки (IDE)
Оберіть один із варіантів:

Варіант А: Visual Studio 2022 (Community, Professional або Enterprise).
При встановленні у Visual Studio Installer необхідно обрати навантаження ".NET desktop development" (Розробка класичних програм .NET).

Варіант Б: JetBrains Rider.
Повноцінна альтернатива Visual Studio, підтримує Windows Forms та .NET.

### 1.4. База даних
Окреме встановлення сервера баз даних не потрібне. Проєкт використовує SQLite, яка працює локально з файлом editor.db.

## 2. Завантаження проєкту

1. Відкрийте термінал (Command Prompt, PowerShell або Git Bash).
2. Перейдіть у папку, де плануєте розмістити проєкт.
3. Виконайте команду:
git clone https://github.com/ВАШ_НІКНЕЙМ/НАЗВА_РЕПОЗИТОРІЮ.git

## 3. Відкриття та відновлення пакетів (Для Visual Studio 2022)
1. Запустіть Visual Studio.
2. Натисніть File -> Open -> Project/Solution.
3. Оберіть файл TextEditor.sln у папці проєкту.
4. Відновлення пакетів NuGet (бібліотек Dapper, System.Data.SQLite) зазвичай відбувається автоматично при першій збірці.
5. Якщо пакети не завантажились: натисніть правою кнопкою миші на рішенні (Solution) у провіднику рішень та оберіть Restore NuGet Packages.

## 4. Запуск програми (З Visual Studio 2022 / Rider)
1. Переконайтеся, що обрано конфігурацію Debug та платформу Any CPU (або x64).
2. Натисніть кнопку Start або клавішу F5.

## 5. Примітки щодо бази даних
Файл бази даних editor.db повинен знаходитися в тій самій директорії, що й виконуваний файл програми (TextEditor.exe).
При компіляції Visual Studio автоматично копіює цей файл у папку виводу (bin/Debug/netX.X), якщо у властивостях файлу editor.db встановлено параметр "Copy to Output Directory" у значення "Copy if newer".
